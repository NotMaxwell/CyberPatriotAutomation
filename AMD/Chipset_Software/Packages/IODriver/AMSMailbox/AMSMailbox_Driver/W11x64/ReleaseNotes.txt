AMS-MailBoxDrv 
===================================

Installation Notes:
-------------------
Install via Windows Device Manager as outlined in the Readme file.


Revision History:
-----------------


5.0.0.1075
Added support for G2 (S5) on getting Soft OFF DASH command.
Added support for AD PWD
Fixed S4 long run Crash
Added support for CSR

4.5.0.1020
Resetting Producer and Consumer Offsets at the beginning and end of KVM.
Added support for AMF KVM supporting 4K resolution using Hardware Accelerated VNC.
Added support for AMS to avoid reboot between installation.

4.5.0.974
Updated KVM Tx Rx processing. Removed Tx polling thread. Changed cleanup to happen as part of IOCTL.

4.5.0.951
Added fix for DHCP timer reset to enable non-Provisioned systems to enter x86 NRC condition.

4.5.0.930
Updated driver code with simplified & uniform logic for passing data MPM->AMS, AMS->MPM, Driver->MPM

4.5.0.866
Added commands for Wi-Fi Sync capabilties.

4.0.0.833
Added commands for User Consent for KVM.

4.0.0.759
Added certification for Windows 24H2

4.0.0.736
Added support for Home / Office Network detection

4.0.0.723
Updated copyright info in inf

4.0.0.685
Added support for Cloud certificate passing from MPM to AMS

3.0.0.635
Separated Keyboard Mouse logic buffers into separate API commands
Maintaining backward compatibility to older Keyboard mouse buffer logic

2.0.0.572
Fixed NCC warnings
Added separate logic for WiFi Port profile

1.0.0.495
