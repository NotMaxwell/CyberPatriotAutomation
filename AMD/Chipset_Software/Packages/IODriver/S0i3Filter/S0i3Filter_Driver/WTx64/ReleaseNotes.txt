Current Release:
-----------------------------------------------------------------------------------------------------
Version : 1.1.0.7
Date    : 04-25-2025

Major Changes to this driver package follows:

1. Driver load is made as optional.
2. Driver will only work when AiM-T and S0i3 is enabled in the system.
2. Added resilience to NDIS callbacks
3. Updated synchronizations.
3. Fixed few crashes observed during validation.
